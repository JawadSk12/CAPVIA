# Candidate Guide

## Starting Your Test
1. Navigate to the test access page
2. Enter your unique access code
3. Read the instructions carefully
4. Click "Begin Test" when ready

## During the Test
- Your progress is saved automatically every 30 seconds
- Use the Previous/Next buttons to navigate questions
- The timer shows remaining time in the top right
- Do not switch browser tabs — this is tracked

## Submitting
1. Review all questions using the progress tracker
2. Click "Submit Test"
3. Confirm submission in the dialog
4. You will see a confirmation page
