# Admin Guide

## Getting Started
1. Login at `/` with admin credentials
2. Navigate to the Admin Dashboard

## Creating a Test
1. Go to **Create Test**
2. Fill in title, description, duration
3. Add questions manually or use AI generation
4. Set question types and difficulty levels
5. Publish the test and share the access code

## Monitoring Live Sessions
- Go to **Dashboard** → Live Sessions
- View candidate progress in real-time
- Red flags indicate suspicious behavior
- Click a session to view behavioral events

## Reviewing Results
1. Go to **Reports**
2. Select a completed test
3. Download individual or bulk reports
