# AI Simulation Engine - Frontend

React + TypeScript + Vite frontend for the AI Simulation Engine assessment platform.

## 🚀 Quick Start

```bash
# Install dependencies
npm install --legacy-peer-deps

# Note: Use --no-bin-links if running on an external USB/exFAT drive to avoid lstat/symlink errors:
# npm install --legacy-peer-deps --no-bin-links

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
